## Restaurant Revenue Analysis

This project analyzes restaurant transaction data to identify factors affecting customer spending and tipping behavior.

### Methods
- Data preprocessing and cleaning
- Log transformation for skewed variables
- Correlation analysis
- T-tests
- One-way ANOVA
- Two-way ANOVA
- Data visualization using seaborn and matplotlib

### Key Findings
- Weekend transactions generate significantly higher revenue.
- Tip amounts are positively correlated with bill amount.
- A statistically significant interaction exists between day and smoker status affecting tip percentage.
